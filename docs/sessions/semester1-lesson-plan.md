# Sellstagram 1학기 수업 계획안
> 마케팅 실습 플랫폼 | 총 15회 | 회당 2시간

---

## 학기 개요

| 항목 | 내용 |
|------|------|
| 대상 | 고등학생 (팀 단위 실습) |
| 수업 횟수 | 15회 |
| 회당 시간 | 2시간 (120분) |
| 학기 목표 | 마케팅 기초 개념 습득 → SNS 마케팅 스킬 → 팀 캠페인 실습 → 발표 |
| 사용 플랫폼 | Sellstagram, ChatGPT/Claude, Canva AI |

---

## 학기 흐름 한눈에 보기

```
1~3회   → 오리엔테이션 & 기초 개념
4~7회   → SNS 스킬 & AI 도구
8~10회  → 팀 캠페인 실습
11~13회 → 심화 스킬
14~15회 → 최종 발표 & 마무리
```
