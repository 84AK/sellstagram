# Sellstagram 2학기 수업 계획안
> 마케팅 실습 플랫폼 | 총 14회 | 회당 2시간

---

## 학기 개요

| 항목 | 내용 |
|------|------|
| 대상 | 고등학생 (팀 단위 실습) |
| 수업 횟수 | 14회 |
| 회당 시간 | 2시간 (120분) |
| 학기 목표 | 심화 마케팅 전략 → 고급 스킬 → 최종 프로젝트 발표 |
| 사용 플랫폼 | Sellstagram, Claude / GPT-4o, Canva, Runway |
| 전제 조건 | 1학기 커리큘럼 이수 |

---

## 학기 흐름 한눈에 보기

```
16~19회  → 심화 학습 (스토리텔링, 영상, AI, 인플루언서)
20~24회  → 고급 전략 (고객여정, SEO, 뉴스레터, 예산, 크리에이터)
25~27회  → 최종 프로젝트 제작
28~29회  → 최종 발표 & 수료
```
