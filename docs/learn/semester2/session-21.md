## 🎯 학습 목표

- 콘텐츠 퍼널 개념 이해
- Gemini로 통합 캠페인 흐름 설계
- 각 콘텐츠 간 연결 고리 만들기

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 우리 팀 콘텐츠 여정 지도 `워밍업`

Gemini에게 우리가 만든 콘텐츠 목록을 알려주고 '이것들을 캠페인으로 연결하면 어떤 순서로 내보내면 좋을까?'라고 질문해봐요. AI가 즉시 그려주는 캠페인 흐름도 확인!

### 📚 [15 ~ 65분] 통합 마케팅 캠페인 전략 수립 `학습`

인지 단계(포스터) → 흥미 단계(퀴즈) → 참여 단계(Zep) → 구매 단계(랜딩페이지)로 이어지는 콘텐츠 퍼널을 설계해요.

### ✏️ [65 ~ 110분] 캠페인 타임라인 + 크로스 링크 설계 `실습`

어떤 콘텐츠를 언제 올리고, 각 콘텐츠에서 다음 콘텐츠로 유도하는 CTA를 설계해요. '퀴즈 결과 보러 Zep 팝업스토어 방문!' 같은 연결 고리!

### 🎤 [110 ~ 120분] 캠페인 전략 발표 `발표`

팀별 캠페인 전략을 3분 발표. 가장 영리한 연결 고리를 만든 팀에 투표!
