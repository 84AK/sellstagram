## 🎯 학습 목표

- 데이터 기반 실시간 의사결정

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 오늘의 마켓 예보! `워밍업`

선생님이 오늘 시뮬레이션의 특별 조건을 발표해요. '오늘은 링크 콘텐츠 효과 3배!' '인터랙티브 콘텐츠 보너스 포인트!' 조건에 따라 팀 전략이 달라져요.

> 💡 **교사 팁**: 매 시뮬레이션마다 다른 조건을 주면 매번 다른 전략이 필요해서 학생들이 더 깊이 생각하게 돼요.

### ✏️ [15 ~ 95분] 시뮬레이션 2차 대회 `실습`

마켓 오픈! 지금까지 쌓아온 모든 콘텐츠 자산을 활용해 최고의 성과를 내봐요. 추가 콘텐츠 업로드, 댓글 관리, 공유 활동 전부 전략적으로!

### ✅ [95 ~ 120분] 2차 대회 결과 + 최종 순위 발표 `마무리`

팀별 성과 발표! 1차 대회 대비 어떻게 성장했는지 비교해봐요. 가장 큰 성장을 이룬 팀 특별 어워즈!
