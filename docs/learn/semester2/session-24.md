## 🎯 학습 목표

- 전체 캠페인 데이터 종합 분석
- 실제 인스타그램 최종 성과 공유
- Gemini로 인사이트 리포트 작성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 데이터 배틀! `워밍업`

어떤 팀의 최종 성과가 가장 좋을지 다른 팀들이 예측해봐요. 예측 후 실제 데이터 공개 → 예측과 차이가 나는 이유 분석!

### ✏️ [15 ~ 75분] Gemini로 최종 인사이트 리포트 작성 `실습`

셀스타그램 데이터와 실제 인스타그램 데이터를 Gemini에게 주고 '가장 성과가 좋은 콘텐츠 유형은? 그 이유는? 앞으로의 전략 제안은?'이라고 물어봐요.

### 🎤 [75 ~ 110분] 팀별 인사이트 공유 `발표`

Gemini가 작성한 리포트를 팀이 직접 검토하고 수정해요. '이 분석이 맞아, 틀려'라고 판단하는 것 자체가 마케터의 역량!

### ✅ [110 ~ 120분] 최종 리포트 셀스타그램 업로드 `마무리`

우리 팀 성과 리포트를 요약한 카드뉴스 형태로 셀스타그램에 올려요.
