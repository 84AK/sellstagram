# 2학기 25회차 — 포트폴리오 완성하기

|------|------|
| 학기/회차 | 2학기 25회차 |
| 테마 | 프로젝트 |
| 난이도 | ⭐⭐ 보통 |
| AI 도구 | Gemini |

## 🎯 학습 목표

- 1년간 제작한 모든 콘텐츠 정리
- Gemini로 개인 성장 회고 작성
- 포트폴리오 목차 완성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 내가 만든 콘텐츠 중 최애는? `워밍업`

1년 동안 만든 콘텐츠 중 가장 자랑스러운 것 1개를 골라요. 왜 그게 최애인지 30초 발표. 이게 포트폴리오의 표지가 될 수도 있어요!

### ✏️ [15 ~ 65분] 포트폴리오 큐레이션 `실습`

포스터, 카드뉴스, 영상, 퀴즈, 음악, Zep, AI 인플루언서... 각 카테고리에서 대표작 1개씩 선정해요. 선정 기준: 가장 잘 만든 것 or 가장 배운 게 많은 것.

### ✏️ [65 ~ 110분] Gemini로 성장 스토리 작성 `실습`

'나는 1년 전 어떤 마케터였고, 지금은 어떻게 달라졌는가'를 Gemini와 함께 작성해봐요. AI가 초안을 써주면 본인이 수정해 진짜 내 이야기로 만들어요.

### ✅ [110 ~ 120분] 포트폴리오 표지 공유 `마무리`

Canva로 만든 포트폴리오 표지를 셀스타그램에 올려요.
