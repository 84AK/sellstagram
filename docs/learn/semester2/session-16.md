## 🎯 학습 목표

- 사운드 브랜딩 개념 이해
- Gemini로 징글 가사 작성
- Suno AI로 브랜드 음악 완성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 내 상품 CM송 들어봐! `워밍업`

Gemini로 징글 가사를 2분 만에 쓰고, Suno AI에 붙여넣으면 30초 후 실제 음악이 완성돼요. 교실에서 AI가 만든 음악이 흘러나오는 순간! 팀마다 다른 장르(발라드/힙합/EDM)로 도전.

> 💡 **교사 팁**: 이 활동만으로도 학생들이 완전히 집중해요. 각 팀 음악을 순서대로 틀어주고 가장 인상적인 CM송에 투표하면 더욱 생동감 있어요.

> 🤖 **AI 프롬프트**:
```
[상품명] 브랜드 CM송 가사를 써줘. 장르는 [발라드/힙합/EDM 중 선택]으로, 30초 분량(약 4~6줄)이면 좋겠어. 후렴구에 상품 이름이 자연스럽게 들어가고, 듣는 사람이 기억하기 쉽게 반복 구조로 만들어줘.
```

### 📚 [15 ~ 55분] 사운드 브랜딩 사례 분석 `학습`

맥도날드 '파라파파파', 삼성 부팅음, 카카오톡 알림음처럼 소리만 들어도 브랜드가 떠오르는 사례를 분석해요. 왜 소리가 기억에 남을까?

### ✏️ [55 ~ 110분] 팀 브랜드 음악 정식 제작 `실습`

인트로에서 즉흥으로 만든 것을 다듬어 정식 브랜드 음악으로 완성해요. 가사, 장르, 분위기를 팀이 함께 결정하고 Suno로 여러 버전을 만들어봐요.

### ✅ [110 ~ 120분] 브랜드 음악 셀스타그램 공유 `마무리`

완성한 Suno 링크를 셀스타그램에 올려요. 가장 브랜드에 잘 맞는 팀 음악 투표!
