## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 우리 팀 콘텐츠 우주 지도 `워밍업`

포스터, 카드뉴스, 영상, 퀴즈, 음악, Zep... 우리가 만든 것들을 칠판에 쭉 나열해봐요. 이 많은 것들을 다 만들었다는 성취감이 가장 좋은 동기부여!

### 📚 [15 ~ 65분] 실제 SNS 채널 중간 성과 점검 `학습`

실제 인스타그램 계정의 팔로워 수, 게시물 별 조회수, 저장 수를 팀별로 공유해요. Gemini로 '왜 이 게시물이 잘됐을까?' 분석 리포트 작성.

### ✏️ [65 ~ 110분] 모든 콘텐츠 링크 셀스타그램 게시 `실습`

Tally 퀴즈 링크, Suno 음악 링크, Zep 팝업스토어 링크를 각각 셀스타그램 게시물로 올려요. 셀스타그램이 진짜 콘텐츠 허브가 되는 순간!

### ✅ [110 ~ 120분] 콘텐츠 포트폴리오 현황 공유 `마무리`

팀별 콘텐츠 수와 종류를 집계해봐요. '가장 다양한 형식의 콘텐츠를 만든 팀' 중간 순위 발표!
