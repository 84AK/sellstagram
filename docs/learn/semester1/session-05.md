## 🎯 학습 목표

- 시각 마케팅의 원칙 이해
- Canva AI로 광고 포스터 완성
- AI 생성 vs 직접 제작 비교 체험

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: AI vs 내가 만든 포스터 `워밍업`

Canva AI에 상품명만 입력하고 30초 만에 자동 생성된 포스터를 전체 화면에 띄워요. '이거보다 내가 더 잘 만들 수 있어!' 경쟁심 자극 후 메인 활동 시작!

> 💡 **교사 팁**: AI 생성 포스터를 저장해두고, 수업 끝에 학생 작품과 나란히 비교하면 성취감이 극대화돼요.

### 📚 [15 ~ 55분] 좋은 광고 포스터 해부하기 `학습`

실제 유명 광고 포스터 3개를 분석해요. 시선이 가는 순서, 색 대비, 카피 위치 등 레이아웃의 비밀을 찾아봐요.

### ✏️ [55 ~ 105분] 팀 상품 포스터 제작 `실습`

브랜드 컬러와 슬로건을 활용해 우리 상품 광고 포스터를 Canva AI로 만들어요. AI를 보조 도구로 쓰되 최종 완성은 우리 손으로!

### ✅ [105 ~ 120분] 포스터 업로드 + 비교 감상 `마무리`

완성한 포스터를 셀스타그램에 올리고 수업 시작 때 AI가 자동으로 만든 것과 비교해봐요. 어떤 게 더 끌리나요?
