## 🎯 학습 목표

- 다른 팀 콘텐츠를 객관적으로 리뷰
- 우리 팀 약점 파악 + 개선 계획 수립
- 2학기 도전 콘텐츠 방향 결정

## 🗓️ 활동 계획

### 🔥 [0 ~ 20분] 🔥 인트로: 내가 고객이라면? `워밍업`

다른 팀의 콘텐츠를 실제 소비자 입장에서 봐요. '이 상품 사고 싶어졌어?' 라는 기준으로 솔직한 1차 반응 공유!

### ✏️ [20 ~ 70분] 크로스 팀 피드백 세션 `실습`

각 팀이 다른 팀의 콘텐츠를 리뷰해요. 잘된 점 2가지 + 개선점 1가지. Gemini로 피드백을 더 구체적으로 다듬어봐요.

### 📚 [70 ~ 110분] 2학기 도전 계획 수립 `학습`

1학기 성과와 피드백을 바탕으로 2학기에 도전할 새로운 콘텐츠(Suno 음악, Zep 메타버스 등)를 결정하고 팀 계획서를 작성해요.

### ✅ [110 ~ 120분] 2학기 도전 선언 `마무리`

팀별로 '2학기에 우리가 도전할 것'을 셀스타그램에 업로드해요.
