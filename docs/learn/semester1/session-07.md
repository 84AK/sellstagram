# 1학기 7회차 — 상품 랜딩페이지 만들기

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 7회차 |
| 테마 | 스킬 |
| 난이도 | ⭐⭐ 보통 |
| AI 도구 | Gemini · Canva AI |

## 🎯 학습 목표

- 랜딩페이지의 목적과 구성 요소 이해
- CTA(행동 유도) 버튼 기획
- Canva로 상품 소개 웹페이지 완성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 이 랜딩페이지 뭐가 문제야? `워밍업`

잘 만든 랜딩페이지 vs 못 만든 랜딩페이지 2개를 나란히 보여줘요. 팀별로 30초 안에 문제점을 찾아 발표! 폰트/CTA/이미지 위치 등을 직접 찾으면서 오늘 수업 포인트를 자연스럽게 학습.

### 📚 [15 ~ 55분] 좋은 랜딩페이지 구조 분석 `학습`

히어로 이미지 → 핵심 혜택 → 상세 설명 → 후기 → CTA 버튼. Gemini로 우리 상품에 맞는 랜딩페이지 카피를 작성해요.

### ✏️ [55 ~ 105분] Canva로 상품 소개 페이지 제작 `실습`

Canva 웹사이트 기능을 활용해 상품 소개 랜딩페이지를 만들어요. 완성 후 링크를 발급해 팀원과 공유해봐요.

> 💡 **교사 팁**: Canva 웹사이트 기능은 무료로 링크 공유가 가능해요. 실제로 친구에게 링크를 보내고 반응을 보는 것만으로도 큰 동기부여가 돼요.

### ✅ [105 ~ 120분] 랜딩페이지 링크 셀스타그램 공유 `마무리`

완성한 랜딩페이지 링크를 셀스타그램 게시물에 올려요. '클릭해봐!' 유도 문구와 함께!
