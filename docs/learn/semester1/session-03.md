# 1학기 3회차 — 우리 고객은 누구야?

> 타겟 고객 설정 + Gemini 페르소나 제작

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 3회차 |
| 테마 | 개념 |
| 난이도 | ⭐ 쉬움 |
| AI 도구 | Gemini |

## 🎯 학습 목표

- 타겟 마케팅의 중요성 이해
- Gemini로 고객 페르소나 2명 작성
- 페르소나 기반 메시지 차이 체험

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 이 광고, 몇 살이 살까? `워밍업`

실제 유명 브랜드 광고 3개를 보여주고 팀별로 '이 광고의 타겟 연령/성별/관심사'를 30초 안에 예측해요. 정답 공개 후 왜 그렇게 타겟을 잡았는지 이야기해요.

> 💡 **교사 팁**: 나이키, 올리브영, 다이슨처럼 타겟이 뚜렷이 다른 브랜드 광고를 준비하세요.

### ✏️ [15 ~ 60분] Gemini로 페르소나 2명 만들기 `실습`

우리 상품을 살 것 같은 사람 2명을 Gemini와 함께 만들어요. 이름, 나이, 직업, 관심사, SNS 사용 습관, 구매 결정 방식까지 상세하게!

> 💡 **교사 팁**: 프롬프트 예시: '스마트워치를 구매할 가능성이 높은 20대 직장인 페르소나를 이름, 나이, 직업, 관심사, SNS 습관 포함해서 만들어줘'

> 🤖 **AI 프롬프트**:
```
[상품명]을 구매할 가능성이 높은 타겟 페르소나 2명을 만들어줘. 각각 이름, 나이, 직업, 관심사, 자주 쓰는 SNS, 구매 결정 방식을 포함해서 작성해줘.
```

### 📚 [60 ~ 100분] 페르소나별 다른 광고 카피 써보기 `학습`

같은 상품인데 페르소나 A에게 보내는 카피 vs 페르소나 B에게 보내는 카피를 각각 써봐요. 얼마나 다른지 비교!

### ✅ [100 ~ 120분] 페르소나 카드 게시물 업로드 `마무리`

우리 팀 페르소나를 소개하는 게시물을 셀스타그램에 올려요.
