# 1학기 4회차 — 브랜드 아이덴티티 만들기

> 로고 컨셉 · 컬러 · 슬로건 설계

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 4회차 |
| 테마 | 스킬 |
| 난이도 | ⭐ 쉬움 |
| AI 도구 | Gemini · Canva AI |

## 🎯 학습 목표

- 브랜드 아이덴티티 개념 이해
- 컬러 심리학 기초 습득
- 팀 브랜드 컬러/슬로건 완성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 10초 브랜드 컬러 테스트 `워밍업`

Gemini에게 '우리 상품에 맞는 브랜드 컬러 5가지를 추천하고 각각의 심리적 의미를 설명해줘'라고 입력해요. 즉시 나오는 결과를 보며 '컬러도 마케팅이구나!' 체감!

> 🤖 **AI 프롬프트**:
```
[상품명] 브랜드에 어울리는 메인 컬러 5가지를 추천해줘. 각 컬러의 심리적 의미와 이 브랜드에 왜 맞는지 이유도 설명해줘.
```

### ✏️ [15 ~ 65분] 브랜드 컬러 · 폰트 · 슬로건 결정 `실습`

Gemini의 추천을 참고해 우리 팀만의 브랜드 컬러 2가지, 폰트 스타일, 슬로건을 완성해요. 선택한 이유를 정리해두면 나중에 발표 때 쓸 수 있어요.

### ✏️ [65 ~ 110분] Canva AI로 브랜드 로고 제작 `실습`

결정한 컬러와 슬로건을 바탕으로 Canva AI에서 팀 로고를 만들어요. AI Magic Design 기능을 활용해봐요.

> 💡 **교사 팁**: 로고는 나중에 카드뉴스, 포스터, 영상 등 모든 콘텐츠에 들어가니 완성도를 높이도록 충분한 시간을 줘요.

### ✅ [110 ~ 120분] 브랜드 카드 공유 `마무리`

컬러 + 슬로건 + 로고를 담은 브랜드 카드를 셀스타그램에 올려요.
