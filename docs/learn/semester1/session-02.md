# 1학기 2회차 — 내 상품을 골라라!

> 스토어 상품 구매 + Gemini 상품 분석

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 2회차 |
| 테마 | 오리엔테이션 |
| 난이도 | ⭐ 쉬움 |
| AI 도구 | Gemini |

## 🎯 학습 목표

- 스토어에서 홍보할 상품 구매
- Gemini로 상품 분석 리포트 작성
- 상품의 강점/약점 파악

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: Gemini야, 광고 멘트 만들어줘! `워밍업`

상품 이름만 Gemini에 입력하면 30초 만에 광고 카피 3가지가 나와요. 팀별로 다른 상품으로 즉시 생성하고 가장 멋진 카피 투표!

> 💡 **교사 팁**: 프롬프트 예시를 칠판에 적어두세요: '스마트워치를 홍보하는 인스타그램 광고 카피 3가지 써줘. 20대 타겟.'

> 🤖 **AI 프롬프트**:
```
[상품명]을 홍보하는 인스타그램 광고 카피 3가지를 써줘. 각각 다른 감성으로: 감성적, 유머러스, 정보 중심. 타겟은 20대야.
```

### ✏️ [15 ~ 55분] 셀러 상점에서 상품 구매 `실습`

셀스타그램 스토어를 둘러보고 이번 학기 내내 홍보할 상품을 구매해요. 구매 전 팀원들과 상의하고 어떤 상품이 마케팅하기 좋을지 이야기해봐요.

> 💡 **교사 팁**: 상품마다 카테고리와 가격이 달라요. 팀별로 다른 상품을 선택하도록 유도하면 나중에 경쟁이 더 재미있어요.

### 📚 [55 ~ 100분] Gemini로 상품 분석 리포트 작성 `학습`

구매한 상품을 Gemini에게 소개하고 '이 상품의 강점, 약점, 주요 타겟, 경쟁 상품'을 분석해달라고 요청해봐요. AI 답변을 팀이 함께 검토하고 수정해요.

> 🤖 **AI 프롬프트**:
```
[상품명]의 강점 3가지, 약점 2가지, 주요 타겟 고객, 주요 경쟁 상품을 마케팅 관점에서 분석해줘. 각 항목을 간결하게 정리해줘.
```

### ✅ [100 ~ 120분] 상품 소개 게시물 업로드 `마무리`

분석 리포트를 바탕으로 '이 상품을 선택한 이유'를 담은 첫 번째 상품 소개 게시물을 셀스타그램에 올려요.
