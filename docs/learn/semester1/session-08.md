# 1학기 8회차 — 우리 상품 퀴즈 만들기

> Gemini + Tally.so 인터랙티브 퀴즈

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 8회차 |
| 테마 | 스킬 |
| 난이도 | ⭐⭐ 보통 |
| AI 도구 | Gemini · Tally.so |

## 🎯 학습 목표

- MBTI/추천형 퀴즈 기획 방법 이해
- Gemini로 조건 분기 시나리오 설계
- Tally.so로 퀴즈 구현 + 링크 공유

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 마케터 MBTI 퀴즈 풀어보기 `워밍업`

선생님이 미리 Tally로 만들어 둔 '마케터 유형 테스트' 링크를 공유해요. 학생 전원이 직접 풀고 결과 화면을 친구들과 비교! '이런 걸 우리 상품으로 만들 수 있겠는데?'로 자연스럽게 연결.

> 💡 **교사 팁**: 1회차에 사용한 테스트와 다른 새 버전을 만들어두면 더 좋아요. 결과 화면에 이미지를 넣으면 시각적 효과 UP.

### 📚 [15 ~ 65분] Gemini로 퀴즈 시나리오 설계 `학습`

우리 상품과 연결되는 MBTI형 or 추천형 퀴즈를 기획해요. Gemini에게 질문 5개와 각 답변에 따른 결과 분기를 설계해달라고 요청해봐요.

> 💡 **교사 팁**: 예시 프롬프트: '스마트워치를 추천하는 MBTI 퀴즈를 만들어줘. 질문 5개, 각 질문에 보기 4개, 결과는 3가지 타입으로 나눠줘.'

> 🤖 **AI 프롬프트**:
```
[상품명]을 추천하는 MBTI형 퀴즈를 만들어줘. 질문 5개, 각 질문마다 보기 4개, 최종 결과는 3가지 타입으로 나눠줘. 각 결과 타입마다 이름, 설명(2줄), 이 상품이 왜 맞는지 이유도 써줘.
```

### ✏️ [65 ~ 110분] Tally.so로 퀴즈 구현 `실습`

Gemini가 설계한 시나리오를 Tally.so에서 실제로 만들어요. 조건 분기, 결과 페이지, 공유 링크까지 완성!

### ✅ [110 ~ 120분] 퀴즈 링크 셀스타그램 업로드 `마무리`

완성한 퀴즈 링크를 셀스타그램에 공유하고 다른 팀 퀴즈를 서로 풀어봐요.
