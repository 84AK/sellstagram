# 1학기 15회차 — 1학기 마무리 + 2학기 예고

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 15회차 |
| 테마 | 복습 |
| 난이도 | ⭐ 쉬움 |

## 🎯 학습 목표

- 1학기 핵심 개념 정리
- 방학 중 실제 인스타그램 운영 미션
- 2학기 콘텐츠 미리 기획

## 🗓️ 활동 계획

### 📚 [0 ~ 40분] 1학기 배운 것 한눈에 정리 `학습`

포스터, 카드뉴스, 쇼츠, 퀴즈, AI 인플루언서까지 1학기 콘텐츠 여정을 함께 돌아봐요. 가장 어려웠던 것 vs 가장 재미있었던 것 공유.

### ✅ [40 ~ 80분] 방학 미션: 실제 인스타그램 운영 `마무리`

방학 동안 팀 AI 인플루언서 계정에 주 1회 이상 게시물 업로드 미션! 방학이 끝나면 팔로워 수와 가장 반응 좋은 게시물을 발표해요.

> 💡 **교사 팁**: 방학 미션이 부담스럽지 않도록 '딱 1개라도 올리면 OK'라는 기준을 제시하세요. 부담 없는 도전이 지속적 참여를 만들어요.

### ✅ [80 ~ 120분] 2학기 프리뷰 + 자유 실습 `마무리`

2학기에 배울 Suno 음악, Zep 메타버스를 잠깐 미리 보여줘요. '이런 것도 할 수 있어?' 기대감 UP! 남은 시간은 셀스타그램 자유 실습.

> 💡 **교사 팁**: Suno에서 샘플 음악 1개를 미리 만들어 틀어주고, Zep 공간 스크린샷을 보여주면 효과적이에요.
