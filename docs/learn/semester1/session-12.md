## 🎯 학습 목표

- 지금까지 콘텐츠 성과 데이터 분석
- 셀스타그램 vs 실제 인스타그램 반응 비교
- 시뮬레이션 1차 대회 참가

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 예측 배틀! `워밍업`

지금까지 올린 콘텐츠 중 '어떤 게 가장 좋아요가 많을까?' 팀별로 예측 후 실제 데이터 공개! 예측과 결과의 차이에서 마케팅 인사이트가 나와요.

### 📚 [15 ~ 55분] 셀스타그램 vs 실제 인스타그램 비교 `학습`

두 플랫폼에 올린 같은 콘텐츠의 반응을 비교해요. 가상 시뮬레이션과 실제 사람의 반응은 얼마나 다를까? Gemini로 인사이트 리포트를 작성해요.

### ✏️ [55 ~ 105분] 시뮬레이션 1차 대회 `실습`

선생님이 마켓 시뮬레이션을 오픈해요! 지금까지 올린 콘텐츠로 경쟁 시작. 추가 콘텐츠를 업로드해 성과를 높여봐요.

> 💡 **교사 팁**: 시뮬레이션 조건을 미리 발표하지 말고 시작할 때 공개하면 긴장감이 UP. 예: '오늘은 이미지 콘텐츠 효과 2배!'

### ✅ [105 ~ 120분] 1차 대회 결과 발표 `마무리`

팀별 성과 순위 공개! 잘된 팀의 전략을 함께 분석하고 배울 점을 이야기해요.
