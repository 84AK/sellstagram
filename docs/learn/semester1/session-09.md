## 🎯 학습 목표

- 숏폼 영상 후킹 기법 이해
- Gemini로 15초 스크립트 작성
- Vrew AI로 자동 자막 영상 완성

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: Gemini야, 15초 광고 스크립트 써줘! `워밍업`

상품명과 타겟만 입력하면 Gemini가 즉시 15초 쇼츠 스크립트를 작성해요. 팀 대표가 실제로 읽어보면서 '이걸 영상으로 만들면 되겠다!' 동기 부여. 가장 임팩트 있는 스크립트 팀 투표!

> 🤖 **AI 프롬프트**:
```
[상품명]을 홍보하는 15초 숏폼 영상 스크립트를 써줘. 타겟은 [타겟층]이야. 첫 3초에 시선을 확 잡는 후킹 멘트로 시작하고, 자막으로 읽기 좋게 한 문장씩 끊어줘.
```

### 📚 [15 ~ 55분] 숏폼 영상 후킹 전략 `학습`

첫 3초에 시선을 잡는 방법: 충격적인 질문, 반전 오프닝, 강한 비주얼. 실제 쇼츠 5개의 첫 3초를 분석해봐요.

### ✏️ [55 ~ 105분] Vrew AI로 쇼츠 영상 제작 `실습`

Gemini가 쓴 스크립트를 Vrew AI에 붙여넣으면 AI 음성 + 자동 자막 영상이 완성돼요. 배경음악과 이미지를 추가해 15~30초 영상으로 마무리!

> 💡 **교사 팁**: Vrew AI는 무료로 사용 가능하며 자동 자막이 매우 정확해요. 모바일에서도 작동하니 스마트폰으로 실습해도 좋아요.

### ✅ [105 ~ 120분] 영상 업로드 + AI 반응 확인 `마무리`

완성한 쇼츠 영상을 셀스타그램에 올리고 AI 고객 반응을 함께 봐요.
