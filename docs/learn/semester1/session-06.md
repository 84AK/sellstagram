# 1학기 6회차 — 5컷으로 설득하는 카드뉴스

> Gemini + Canva AI 카드뉴스 제작

| 항목 | 내용 |
|------|------|
| 학기/회차 | 1학기 6회차 |
| 테마 | 스킬 |
| 난이도 | ⭐⭐ 보통 |
| AI 도구 | Gemini · Canva AI |

## 🎯 학습 목표

- 카드뉴스 스토리 구조 이해
- Gemini로 본문 카피 작성
- 5컷 카드뉴스 완성 + 업로드

## 🗓️ 활동 계획

### 🔥 [0 ~ 15분] 🔥 인트로: 카드뉴스 제목 배틀 `워밍업`

같은 상품, 같은 주제로 팀마다 Gemini에게 카드뉴스 제목 5개를 뽑게 해요. 전체 공유 후 '가장 클릭하고 싶은 제목'에 투표! 왜 그 제목이 이겼는지 분석.

> 🤖 **AI 프롬프트**:
```
[상품명]을 소개하는 카드뉴스 제목 5가지를 만들어줘. 각각 다른 스타일로: 궁금증 유발형, 숫자 활용형, 공감형, 혜택 강조형, 도발형.
```

### 📚 [15 ~ 60분] 카드뉴스 5컷 구조 기획 `학습`

1컷: 후킹 제목 / 2~4컷: 핵심 내용 / 5컷: CTA(행동 유도). Gemini와 함께 각 컷의 내용을 기획해요.

> 💡 **교사 팁**: Gemini 프롬프트 예시: '스마트워치를 소개하는 5컷 카드뉴스 구성을 짜줘. 각 컷에 들어갈 제목과 내용을 간단히 써줘.'

> 🤖 **AI 프롬프트**:
```
[상품명]을 소개하는 5컷 카드뉴스 구성을 짜줘. 1컷: 후킹 제목, 2~4컷: 핵심 내용(각 컷 한 가지 포인트), 5컷: 행동 유도 문구. 각 컷에 들어갈 제목과 본문 2줄을 써줘.
```

### ✏️ [60 ~ 105분] Canva AI로 카드뉴스 제작 `실습`

기획한 내용을 바탕으로 Canva에서 5컷 카드뉴스를 디자인해요. 브랜드 컬러와 로고를 일관되게 사용해요.

### ✅ [105 ~ 120분] 카드뉴스 업로드 + 반응 확인 `마무리`

완성한 카드뉴스를 셀스타그램에 올리고 AI 고객 반응을 함께 봐요.
