# Plan: teacher-sim-monitor

> 선생님 대시보드 - 마켓 시뮬레이션 실시간 결과 모니터링

**Created**: 2026-03-11
**Phase**: Plan → Design → Do
**Level**: Dynamic

---

## 1. 배경 및 목표

### 현재 문제
- 선생님이 시뮬레이션을 On/Off 켜고 끌 수 있지만, **누가 어떤 결과를 냈는지 볼 수 없음**
- 각 학생의 시뮬레이션 결과는 `simulation_results` 테이블에 저장되지만 교사 화면에 표시 안됨
- 수업 중 "어떤 팀이 매출을 가장 많이 냈나?" 확인 불가

### 목표
선생님 대시보드에 **시뮬레이션 결과 모니터링 탭** 추가:
- 학생이 "결과 저장" 클릭 시 → 선생님 화면에 **실시간 알림 + 카드** 표시
- 팀별 총매출 집계 리더보드
- 개별 학생 결과 카드 (이름, 게시물, 좋아요/댓글/구매/매출)

---

## 2. 사용자 스토리

| 역할 | 시나리오 | 가치 |
|------|---------|------|
| 선생님 | 시뮬레이션 진행 중 각 학생의 결과를 실시간으로 보고 싶다 | 수업 피드백 즉시 제공 |
| 선생님 | 팀별 총매출을 한눈에 비교하고 싶다 | 경쟁 심화, 수업 활성화 |
| 선생님 | 가장 성과 좋은 게시물을 강조 표시하고 싶다 | 우수 사례 공유 |

---

## 3. 기능 범위

### In Scope
- [ ] 선생님 대시보드에 "시뮬레이션" 탭 추가
- [ ] `simulation_results` Supabase Realtime 구독
- [ ] 실시간 결과 카드 표시 (최신 결과가 위에)
- [ ] 팀별 총매출 집계 리더보드
- [ ] 새 결과 수신 시 토스트 알림

### Out of Scope
- 학생 화면에서 다른 학생 결과 보기
- 시뮬레이션 이벤트 (좋아요/댓글 등) 개별 로그 표시
- 결과 삭제/수정 기능

---

## 4. 기술 제약

- `simulation_results` 테이블이 schema.sql에 없음 → Supabase에는 존재(API로 확인됨)
- `simulation_results`에 Realtime 활성화 필요 (현재 미활성)
- 선생님 페이지는 PIN 인증 기반 → Supabase anon key로 조회 가능
