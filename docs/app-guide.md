# 셀스타그램(Sellstagram) 사용 설명서
> 이 설명서는 선생님과 학생 모두를 위한 쉬운 안내서입니다.

---

## 이 앱이 뭐야? 🤔

**셀스타그램**은 고등학생들이 인스타그램처럼 생긴 앱으로 **마케팅 공부**를 하는 시뮬레이션 게임이에요.

진짜 돈을 쓰거나 진짜 SNS에 올리는 게 아니에요.
앱 안에서 가상의 돈으로 상품을 사고, 광고 게시물을 올리고, AI가 점수를 매겨주는 놀이처럼 공부하는 거예요!

```
비유하자면...
학교에서 하는 "모의 주식 투자"처럼
진짜 돈 없이 마케팅을 연습하는 것!
```

---

## 이 앱에는 누가 있어? 👥

이 앱에는 3가지 역할이 있어요.

| 역할 | 누구? | 하는 일 |
|------|-------|---------|
| 🎓 **선생님** | 수업 진행자 | 미션 등록, 수업 진행, 학생 현황 모니터링 |
| 🧑‍💻 **학생** | 팀별 참여자 | 게시물 올리기, 미션 도전, 포인트 모으기 |
| 🔑 **관리자** | 앱 개발자/운영자 | 서버 설정, 계정 관리 (일반적으로 선생님이 알 필요 없음) |

---

## 전체 흐름 한눈에 보기 🗺️

```
[수업 전] 선생님이 준비
    ↓
    1. 팀 코드 정하기 (A팀, B팀, C팀...)
    2. 미션 등록하기 (/teacher 페이지)
    3. 오늘 수업 회차 확인하기

[수업 중] 학생들이 참여
    ↓
    1. 앱 접속 → 온보딩(처음 1번만)
    2. 팀 코드 입력해서 팀에 합류
    3. 마케팅 게시물 올리기
    4. AI 코치 피드백 받기
    5. 미션 달성 → 자동으로 완료 + 보상!

[수업 후] 선생님이 마무리
    ↓
    1. 팀별 순위 확인
    2. 우수 팀/콘텐츠 소개
```

## 선생님 사용법 🎓

### 1단계: 선생님 대시보드 접속

앱 주소 뒤에 `/teacher` 를 붙여서 접속해요.

```
예: https://sellstagram.vercel.app/teacher
```

처음 접속하면 **4자리 PIN 번호**를 입력하는 화면이 나와요.
PIN 번호는 `.env` 파일에 있는 `TEACHER_PIN` 값이에요. (관리자에게 문의)

---

### 2단계: 선생님 대시보드에서 할 수 있는 것

선생님 대시보드는 탭 3개로 나뉘어요.

#### 📋 수업 탭 (Class)
- 오늘이 몇 회차 수업인지 볼 수 있어요
- ◀ ▶ 버튼으로 이전/다음 회차로 이동
- 오늘 수업에서 학생들이 뭘 해야 하는지 가이드가 나와요

#### 📱 피드 탭 (Feed)
- 학생들이 올린 게시물을 실시간으로 볼 수 있어요
- 어떤 팀이 얼마나 활발한지 한눈에 파악 가능

#### 🎯 미션 탭 (Mission)
- **미션 등록** 여기서 해요!
- 학생들이 도전할 미션을 만들고 ON/OFF 할 수 있어요

---

### 3단계: 미션 등록하기 ✅

> **예, 미션은 선생님이 등록해요!**

미션 탭에서 `+ 새 미션` 버튼을 누르면:
- 미션 제목 입력 (예: "이번 주 게시물 3개 올리기")
- 미션 설명 입력
- 보상 포인트 설정 (숫자로 입력)

등록된 미션은 학생들의 `/missions` 페이지에 바로 나타나요.

미션 옆의 **토글 버튼**으로 미션을 켜고 끌 수 있어요.
- 🟢 ON: 학생들에게 보임
- ⚫ OFF: 학생들에게 안 보임

---

### 4단계: 팀 현황 보기

선생님 대시보드 상단에 팀별 점수판이 있어요:
- 각 팀의 게시물 수
- 각 팀의 총 점수
- 팀원 수

수업 중 프로젝터에 띄워두면 학생들 경쟁심을 자극할 수 있어요!

## 학생 사용법 🧑‍💻

### 1단계: 처음 접속 (온보딩) — 딱 1번만!

앱에 처음 들어오면 **온보딩 화면** 3단계가 나와요.

```
1단계: 나는 어떤 마케터?
    → 크리에이터 / 분석가 / 스토리텔러 / 이노베이터 중 선택

2단계: 내 정보 입력
    → 이름, 팀 코드(A팀/B팀...), 아바타 선택

3단계: 완료!
    → 마케터 프로필 카드 생성
```

> ⚠️ 팀 코드는 선생님이 칠판에 써준 것을 입력하세요!

---

### 2단계: 앱 화면 설명

#### 🏠 홈 (피드)
- 우리 반 학생들이 올린 게시물이 다 보여요
- 다른 팀 게시물에 좋아요 누를 수 있어요
- **[+ 업로드]** 버튼으로 내 게시물 올리기

#### 📅 세션 허브
- 1~29회 수업 커리큘럼이 다 있어요
- 오늘 수업이 뭔지, 다음엔 뭘 배울지 볼 수 있어요
- 각 회차를 누르면 오늘 뭘 해야 하는지 자세히 설명이 나와요

#### 🎯 미션
- 선생님이 만든 미션 목록이 나와요
- 미션 진행률이 막대 그래프로 보여요
- 미션을 달성하면 **자동으로 완료 처리 + 보상 지급!**
  (오른쪽 아래에 초록색 팝업이 떠요 🎉)

#### 🛒 셀러 상점
- 가상 상품 목록이 있어요 (스마트워치, 파우치, 이어폰...)
- 가상 돈(₩)으로 상품을 "구매"해요
- 구매한 상품이 내 마케팅 아이템이 돼요

#### 📚 학습 자료
- 마케팅 개념 카드 (4P, AIDA 등)
- AI 도구 사용법
- 단계별 튜토리얼

#### 👤 프로필
- 내 포인트, 팀, 랭크 확인
- 내가 올린 게시물 목록

---

### 3단계: 게시물 올리기

홈 화면에서 **[+ 업로드]** 버튼을 누르면 팝업이 떠요.

```
1. Photo Post / Video Reels 중 선택
2. 이미지 업로드 (드래그 or 클릭)
3. 광고 문구(캡션) 작성
4. 해시태그 입력
5. [AI 분석하기] 버튼 → AI 코치 피드백 받기 (선택)
6. [실세계 피드에 게시하기] 버튼 → 업로드 완료!
```

업로드 하면:
- 포인트 +10 XP 자동 지급
- 미션 조건 달성 여부 자동 체크
- 미션 완료 시 추가 보상 지급!

---

## 포인트 & 보상 시스템 💰

| 행동 | 얻는 것 |
|------|---------|
| 게시물 1개 업로드 | +10 XP |
| 미션 모드로 게시물 업로드 | +20 XP |
| 미션 완료 | 미션별 보상 포인트 |

포인트가 쌓이면 랭크가 올라가요:
- Beginner → Marketer → Pro → Elite → Legend

---

## 미션 자동 완료란? 🤖

선생님이 미션을 만들 때 **미션 타입**이 자동으로 설정돼요.

현재 지원하는 미션 타입:

| 타입 | 예시 | 자동 완료 조건 |
|------|------|----------------|
| 게시물 수 | "게시물 3개 올리기" | 게시물 3개 업로드하면 자동 완료 |
| 매출 | "매출 500만원 달성" | 게시물 판매 합계가 500만원 넘으면 완료 |
| 인게이지먼트 | "인게이지먼트 10% 게시물 3개" | 해당 게시물 수 채우면 완료 |

> 학생이 직접 버튼을 누를 필요 없이, 조건만 채우면 **짠!** 하고 완료돼요.

## 자주 묻는 질문 ❓

**Q: 팀 코드를 틀리게 입력했어요!**
> A: 지금은 온보딩을 다시 하려면 브라우저에서 "사이트 데이터 삭제"를 하면 돼요. (개발자 도구 → Application → Clear Storage)

**Q: 학생이 게시물을 올렸는데 피드에 안 보여요**
> A: Supabase(데이터베이스) 연결이 안 된 경우 본인 화면에만 보일 수 있어요. 관리자에게 DB 연결 여부를 확인하세요.

**Q: 미션이 완료됐는데 보상이 안 왔어요**
> A: 오른쪽 하단 초록 팝업을 확인하세요. 잔액도 자동으로 올라가 있을 거예요. 페이지를 새로고침해도 돼요.

**Q: 선생님 PIN 번호가 뭐예요?**
> A: 관리자(개발자)만 알고 있어요. `.env` 파일의 `TEACHER_PIN` 값이에요.

**Q: 실제 돈이나 진짜 SNS와 연결되나요?**
> A: 전혀요! 모든 것이 앱 안에서만 일어나는 가상 시뮬레이션이에요.

수업 시작 전에 이것만 확인하세요!

```
□ 팀 코드를 칠판에 써두기 (A팀, B팀, C팀...)
□ /teacher 로 접속해서 PIN 입력 확인
□ 오늘 수업 회차가 맞는지 확인
□ 이번 수업에 맞는 미션 ON 상태인지 확인
□ 학생들 인터넷 연결 확인
□ 프로젝터에 선생님 대시보드 팀 현황 띄우기 (선택)
```
