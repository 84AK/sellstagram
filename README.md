# 📸 셀스타그램 (Sellstagram) - 2026 마케팅 시뮬레이션 플랫폼

> **"Z세대를 위한, Z세대에 의한, 실감나는 마케팅 실습 SNS"**

셀스타그램(Sellstagram)은 학생들이 가상의 상품을 주제로 콘텐츠를 기획, 업로드하고, AI가 생성한 가상 고객들의 실시간 반응과 매출 데이터를 통해 마케팅 감각을 익힐 수 있는 에듀테인먼트 플랫폼입니다.

---

## 🚀 주요 기능 (Core Features)

### 1. 하이엔드 소셜 피드 (Social Feed)
- **Instagram Style**: 글래스모피즘 2.0 디자인이 적용된 정교한 게시물 피드.
- **Short-form Video**: 틱톡/릴스 스타일의 세로형 비디오 자동 재생 및 인터랙션.
- **Story Bar**: 몰입감 있는 사용자 경험을 위한 상단 스토리 UI.

### 2. 가상 커머스 엔진 (Virtual Commerce)
- **Seller Shop**: 마케팅할 상품을 탐색하고 보유 자산으로 직접 구매하는 프로세스.
- **Dynamic Simulation**: 캡션, 태그, 비주얼 퀄리티에 따라 실시간으로 변하는 판매 성과 시뮬레이션.
- **Wallet & Inventory**: 실습을 통해 얻은 수익으로 더 가치 있는 상품을 소싱하는 경제 구조.

### 3. Gemini 3.1 Pro 기반 AI 엔진 (AI Powered)
- **AI Coaching**: 세계적인 마케팅 전문가 페르소나가 제공하는 날카로운 게시물 분석 및 조언.
- **Virtual Personas**: 4가지 성격의 가상 고객 페르소나들이 남기는 생생한 실시간 댓글 반응.
- **Vision Analysis**: 업로드 전 시각적 요소(색감, 구도)를 미리 분석하여 성과 예측.

### 4. 에듀케이션 대시보드 (Dashboard)
- **Real-time Insights**: 매출 현황, ROAS, 도달 범위 등 핵심 마케팅 지표 시각화.
- **Weekly Missions**: 주차별 미션을 통해 단계별 성취감 제공.
- **Team Workspace**: 팀별 랭킹과 활동 로그를 통한 협업 및 경쟁 유도.

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend**: Next.js 16 (App Router), Tailwind CSS V4, Lucide-react
- **State Management**: Zustand (Global Game State)
- **AI Models**: Google Gemini 3.1 Pro, Gemini 1.5 Flash
- **Design**: 2026 Glassmorphism 2.0, Bento Grid Layout
- **Animations**: Framer Motion, CSS Scroll-driven Animations

## 📦 시작하기 (Getting Started)

1. **환경 변수 설정**: `.env.local` 파일에 Gemini API 키를 추가하세요.
   ```env
   NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
   ```
2. **패키지 설치**:
   ```bash
   npm install
   ```
3. **개발 서버 실행**:
   ```bash
   npm run dev
   ```
