# 📝 셀스타그램 (Sellstagram) 개발 로그 (Development Log)

## 📅 작성일: 2026-03-01 (Phase 4 완료)

### 1. AI 엔진 고도화 내용 (AI Engine Enhancements)
- **가상 고객 페르소나 시스템**: `p1`~`p4`까지 서로 다른 연령대와 관심사, 말투를 가진 4명의 핵심 페르소나를 정의했습니다 (`personas.ts`).
- **멀티 댓글 반응 생성**: Gemini 3.1 Pro를 사용하여 게시물 내용에 적합한 다량의 실시간 댓글을 JSON 형태로 생성하고 파싱하는 로직을 구축했습니다 (`reactions/route.ts`).
- **시각적 성과 분석(Vision)**: 게시물의 캡션과 이미지(모의) 데이터를 분석하여 마케팅 관점의 구도, 색감, 타겟 적합성을 코칭해주는 Vision 로직을 추가했습니다 (`vision.ts`).
- **업로드 실시간 미리보기**: `UploadModal`에서 게시물을 올리기 전 AI가 즉각적으로 예상 성과를 브리핑해주는 UX를 구현했습니다.

### 2. 수정 사항 및 해결된 에러 (Fixes & Resolved Issues)
- **JSON 파싱 안정성**: AI 응답에 포함된 마크다운 기호를 제거하고 순수 JSON 리스트만 추출하는 정규식/슬라이싱 로직을 강화했습니다.
- **댓글 애니메이션**: 다량의 댓글이 한꺼번에 나타나지 않고 순차적으로 페이드인되는 애니메이션 효과를 적용했습니다.
- **Vision 로직 통합**: Gemini 1.5 Flash를 Vision 전용 모델로 설정하여 빠른 분석 속도를 확보했습니다.

### 3. 기술 스택 자산 (AI Assets)
- **Model**: Gemini 3.1 Pro (Coaching), Gemini 1.5 Flash (Vision/Reaction)
- **Prompts**: 페르소나 맞춤형 `PEOPLE_REACTIONS` 및 마케팅 전문 `VISION_ANALYSIS` 프롬프트 구축 완료.

### 4. 신규 기능: 가이드 시스템 및 실습 컨텍스트 (Guide & Practice Context)
- **프리미엄 가이드 모달**: '가이드' 버튼 클릭 시 벤토 그리드 스타일의 시뮬레이션 안내 모달이 나타납니다 (`GuideModal.tsx`).
- **실습 모드(Mission Mode)**: '실습 시작하기' 버튼을 통해 업로드 모달을 열면, 상단에 'WEEKLY MISSION MODE' 배지가 표시되어 일반 게시물 생성과 구분됩니다.
- **포맷 토글 (Photo/Video)**: 인스타그램 스타일의 사진 게시물과 틱톡/릴스 스타일의 영상 게시물을 선택하여 업로드할 수 있는 토글 기능을 추가했습니다.
- **실시간 데이터 저장**: 작성한 게시물은 전역 상태(`useGameStore`)에 즉시 저장되어 메인 피드에 실습 결과가 바로 반영됩니다.
